FCrackZip (.zip Cracker):
sudo apt install fcrackzip
Usage: fcrackzip -v -u -D -p /usr/share/wordlists/rockyou.txt nameof.zip
v --> Verbose
u --> Unzipping
-D --> Dictionary Attack
-p --> Using Word List
